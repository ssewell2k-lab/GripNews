Archive of the GripNews sites, pulled from the grip0s.com cPanel hosting account on 2026-07-05 during decommission.

- `gripnews.uk/` — main GripNews PHP site (news, blog, signals API)
- `auth.gripnews.uk/` — auth portal
- `cp.gripnews.uk/` — control panel subdomain
- `vl.gripnews.uk/` — VL subdomain (wiki/sitemap/API)

Secrets removed before publishing: `.env` files, database passwords and internal API keys (replaced with `REDACTED`). MySQL dumps are stored separately in a private archive. Nothing here is running anywhere — archived for reference only.
